# Primavera P6 Lookahead Schedule Layout (PLF)

This repository contains a practical lookahead schedule layout for Oracle Primavera P6.

## File Included
- lookahead_layout.plf → Import as a layout in Primavera P6

## How to Use

1. Open your project in Primavera P6
2. Go to:
   View → Layout → Open
3. Click Import and select the PLF file
4. Apply the layout to your schedule

## What This Layout Shows

The layout is designed to support lookahead scheduling by clearly presenting:

- Recently completed activities
- Ongoing work
- Upcoming activities within the lookahead window
- Late activities requiring attention

It also highlights key fields such as:
- Dates (Start/Finish)
- Remaining Duration
- % Complete
- Variances
- Activity Status
- Total Float
